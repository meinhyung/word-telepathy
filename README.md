# 단어 텔레파시

온라인에서 랜덤으로 만난 두 사람이 같은 단어를 말할 때까지 라운드를 반복하는 웹 게임.

## 실행 방법

1. Node.js 설치 (https://nodejs.org — LTS 버전)
2. 이 폴더에서 터미널을 열고:

```bash
npm install
node server.js
```

3. 브라우저에서 `http://localhost:3000` 접속
4. 혼자 테스트하려면 **브라우저 탭 2개**를 열고 각각 매칭 시작을 누르면 서로 매칭됩니다.

## 게임 규칙

- 매 라운드 5초 안에 단어 1개 입력
- 라운드가 끝나면 5초 동안 서로의 단어 공개 + 생각 시간
- 두 사람이 같은 단어를 적으면 게임 종료 (대소문자/공백 무시하고 비교)
- 단어를 입력하지 않으면 그 라운드는 `(미입력)` 처리

## 구조

| 파일 | 역할 |
|---|---|
| `server.js` | 매칭 대기열, 라운드 타이머, 단어 일치 판정 (Socket.IO) |
| `public/index.html` | 게임 화면 전체 (HTML + CSS + JS) |

## 시간 설정 바꾸기

환경변수로 조절할 수 있습니다 (밀리초 단위):

```bash
ROUND_TIME=8000 REVEAL_TIME=6000 node server.js
```
